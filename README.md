# 🌙 Lunete

Lunete AI é uma inteligência artificial criada com o objetivo de ajudar pessoas que estão passando por momentos difíceis e precisam de alguém para conversar.

O projeto foi pensado especialmente para quem não tem acesso fácil a apoio psicológico, seja por falta de tempo ou recursos. No entanto, é importante deixar claro que a Lunete não substitui psicólogos ou profissionais da área, ela atua apenas como uma companhia temporária para diálogo e suporte emocional leve.

Este projeto está em fase beta e foi desenvolvido com foco em baixo custo. Por isso, não utiliza servidores: todo o processamento acontece diretamente no dispositivo do usuário (computador ou celular), garantindo uma experiência acessível e independente.

A Lunete AI também representa um projeto de exploração técnica e criativa, desenvolvido por __**0megaAD**__ , com o objetivo de testar limites, aprender novas tecnologias e transformar ideias em algo funcional.

⚠️ Aviso:
Lunete AI não substitui acompanhamento psicológico profissional. Em casos mais sérios, procure ajuda especializada.

## 🚀 Funcionalidades

- IA rodando no navegador (WebLLM)
- Múltiplos chats
- Histórico salvo no navegador
- Interface moderna

## 🧠 Tecnologias

- WebLLM
- JavaScript
- HTML/CSS

## ⚠️ Limitações

- Dados salvos apenas localmente
- Não sincroniza entre dispositivos

## ▶️ Como usar

1. Abra o index.html
2. Aguarde a IA carregar
3. Comece a conversar
